# Monolith To Microservices Infrastructure

This repository contains the infrastructure-as-code (IaC) configuration for deploying the ** Monolith To Microservices** using **Terraform** on **AWS**. The infrastructure provisions an EKS Kubernetes cluster, a VPC,  S3 bucket and multiple ECR repositories to support the app's microservices as well as a Postgres Database.
